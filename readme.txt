== Description ==

To get all physical or non physical address google places reviews
[youtube https://www.youtube.com/watch?v=qInOeuLRpAk]

[Online demo](https://www.talkforweb.com.au)

= Plugin Features =

* Fetch all google reviews
* Fetch all non physical address reviews also

== Installation ==

1. Upload contents of compressed file (tfw-reviews) to the `/wp-content/plugins/` directory. 
2. Activate the plugin through the `Plugins` menu in WordPress admin.
3. Create a WordPress page to be used specifically for gathering reviews or testimonials.
4. Go into All Review and insert PB id.
